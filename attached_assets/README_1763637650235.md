# Hospitality Procurement Intelligence Platform

A sophisticated web application prototype for procurement management in the hospitality industry, featuring real-time data visualization, predictive analytics, and actionable insights.

## üöÄ Features

### Core Functionality
- **Data Flow Visualization**: Interactive animation showing raw data transformation through a processing funnel
- **Multi-Filter Analysis**: Dynamic data views for Daily, Weekly, Monthly, and Yearly perspectives
- **Actionable Items Matrix**: 6x5 grid showing procurement categories vs. velocity patterns
- **Real-time Bubbles**: Animated status indicators with color-coded efficiency metrics
- **Detailed Insights Drawer**: Comprehensive product details and analytics
- **KPI Dashboard**: Interactive charts for utilization, cost, and wastage trends

### Visual Design
- **Professional Dark Theme**: Modern gradient backgrounds with glassmorphism effects
- **Responsive Layout**: 3/4 main content area with 1/4 insights sidebar
- **Smooth Animations**: CSS transitions and keyframe animations for enhanced UX
- **Interactive Elements**: Hover effects, blinking bubbles, and smooth transitions

### Data Categories
- **Food & Beverages**: Fresh produce, dairy, beverages, meat, bakery items
- **Housekeeping**: Cleaning supplies, linens, toiletries, amenities
- **Maintenance**: Tools, spare parts, electrical, plumbing, safety equipment
- **Guest Utilities**: Towels, bathrobes, welcome kits, room service items
- **Utilities & Supplies**: Office supplies, IT equipment, furniture, lighting
- **Marketing**: Promotional materials, signage, brochures, digital assets

### Velocity Classifications
- **Fast-moving**: High-frequency items requiring frequent replenishment
- **Medium**: Regular consumption items with moderate turnover
- **Slow**: Low-frequency items with predictable usage patterns
- **Very-slow**: Occasional items with minimal consumption
- **Once in a while**: Rare items used for special occasions

## üö¶ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, for best performance)

### Installation
1. Clone or download the project files
2. Ensure all files are in the same directory:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`

### Running the Application
1. **Direct Browser Access**: Open `index.html` in your web browser
2. **Local Server** (recommended):
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```
3. Navigate to `http://localhost:8000`

## üéÆ Usage Guide

### Starting the Application
1. Click the **"Start Analysis"** button to trigger the data flow animation
2. Watch as raw data flows through the processing funnel
3. Observe the actionable items matrix populate with animated bubbles

### Filtering Data
- Use the filter buttons (Daily, Weekly, Monthly, Yearly) to change time perspectives
- All sections update dynamically based on the selected filter
- Monthly view is the default selection

### Exploring Insights
- Click any bubble in the actionable items matrix
- View detailed product information in the right-side drawer
- Analyze purchase cycles, consumption patterns, and cost metrics

### Search Functionality
- Use the search bar to filter specific products or categories
- Real-time filtering with visual feedback
